# Bifrost

SQL Server data migration tool for .NET 10. Named after the rainbow bridge connecting realms in Norse mythology.

## Setup

```bash
cp config.example.json config.json
cp source-connection.example.json source-connection.json
cp target-connection.example.json target-connection.json
# Edit the three files with your settings
```

## Build

```bash
dotnet build
```

## Commands

### export
Exports tables from the source database to `.sql` files. Use when source and target are on different VPNs.

```bash
bifrost export
bifrost export --config config.json --connection source-connection.json --output ./output
```

### import
Imports `.sql` files generated by `export` into the target database.

```bash
bifrost import
bifrost import --connection target-connection.json --output ./output
```

### direct
Migrates directly from source to target with no intermediate files. Use when both servers are reachable on the same network.

```bash
bifrost direct
bifrost direct --config config.json --source source-connection.json --target target-connection.json
```

## Publish as single exe

```bash
# Windows
dotnet publish -r win-x64 -c Release

# Linux
dotnet publish -r linux-x64 -c Release
```

Output will be in `bin/Release/net10.0/<rid>/publish/bifrost`

## Connection file format

```json
{
  "server": "192.168.1.100",
  "port": 1433,
  "username": "sa",
  "password": "yourpassword",
  "encrypt": false,
  "trustServerCertificate": true
}
```

## Config file format

```json
{
  "databases": [
    {
      "sourceDatabase": "MainAppDB",
      "targetDatabase": "MainAppDB_Staging",
      "tableFilter": "tenant",
      "tenantId": "42",
      "overrides": [
        { "name": "dbo.HugeTable_42", "where": "CreatedAt >= '2024-01-01'" },
        { "name": "dbo.BigTable_42", "query": "SELECT TOP 1000 * FROM [dbo].[BigTable_42] ORDER BY CreatedAt DESC" },
        { "name": "dbo.UnwantedTable_42", "ignore": true }
      ]
    },
    {
      "sourceDatabase": "ReportingDB",
      "targetDatabase": "ReportingDB_Staging",
      "tableFilter": "explicit",
      "tables": [
        "dbo.SmallTable",
        { "name": "dbo.HugeTable", "where": "CreatedAt >= '2024-01-01'" },
        { "name": "dbo.UnwantedTable", "ignore": true }
      ]
    },
    {
      "sourceDatabase": "LookupDB",
      "targetDatabase": "LookupDB_Staging",
      "tableFilter": "all",
      "overrides": [
        { "name": "dbo.LargeAuditLog", "ignore": true }
      ]
    }
  ]
}
```

## tableFilter options

| Value | Description |
|-------|-------------|
| `tenant` | Exports all tables ending with `tenantId` |
| `explicit` | Exports only the listed tables |
| `all` | Exports all tables |

## Per-table override options

| Field | Description |
|-------|-------------|
| `ignore: true` | Skip this table entirely |
| `where: "..."` | Filter rows with a WHERE clause |
| `query: "..."` | Use a fully custom SELECT query |

For `tenant` and `all` filters, overrides go in the `overrides` array.
For `explicit` filter, overrides can be inlined in the `tables` array.
